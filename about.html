About the Game

Who's That Pokémon? is a fun and nostalgic guessing game where players try to identify a Pokémon based on a series of clues. Inspired by the iconic segment from the Pokémon anime, this game challenges your memory and knowledge of the Pokémon universe!

Built with ❤️ by a Pokémon lover, this project is constantly evolving — more features, game modes, and Pokémon are on the way!
